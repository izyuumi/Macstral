# Macstral

Macstral is a macOS menu bar app for hotkey-driven local dictation with onboarding and permission-aware setup.

## Features

- Hold-to-dictate interaction from the menu bar
- On-device transcription pipeline
- First-run onboarding for required system permissions

## Installation

```bash
brew tap izyuumi/tap
brew install --cask macstral
```

## First run

On first launch, complete onboarding and grant permissions for:

- Microphone access
- Accessibility access (for text insertion)
